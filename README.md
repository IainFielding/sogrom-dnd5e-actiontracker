# Sogrom's Action Tracker

A [Foundry VTT](https://foundryvtt.com/) module for the [dnd5e](https://github.com/foundryvtt/dnd5e) system that
adds **Action**, **Bonus Action**, and **Reaction** trackers to character and NPC sheets, so players and the GM can
see at a glance what's been used during a turn.

## Features

- Adds Action, Bonus Action, and Reaction indicators to player and NPC sheets.
- When an action is used it is marked it as used for the current turn.
- Automatically resets at the start of each combatant's turn.
- Choose where the trackers appear on player sheets and NPC sheets independently:
  - **Header** - next to the rest of the sheet header buttons.
  - **Portrait** - a collapsible tray attached to the character/token portrait (player sheets) or a row beneath the
    HP meter (NPC sheets).

## Installation

1. In Foundry's **Add-on Modules** tab, click **Install Module**.
2. Search for "Sogrom's Action Tracker", or paste the manifest URL below:

   ```
   https://github.com/IainFielding/sogrom-dnd5e-actiontracker/releases/latest/download/module.json
   ```

3. Enable the module in your world's **Module Management**.

## Compatibility

- Foundry VTT v13-v14 (verified on v14.364)
- dnd5e system v4+ (verified on 5.3.3)
